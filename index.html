HTML (HyperText Markup Language) is the core language used to structure and display content on the web, defining elements like text, images, links, and page layout so browsers can render websites properly.
Provides the basic structure for all web pages using opening and closing tags like 'p, h1, and div'.
Supports text, images, links, lists, tables, forms, and multimedia elements such as audio and video.
Works with CSS for styling and JavaScript for interactivity.
Uses attributes like src, href, and alt to add extra information and control element behavior.
HTML documents are transferred over the Internet using HTTP Protocol.

India’s national anthem is "Jana Gana Mana," composed originally in Bengali by Rabindranath Tagore on December 11, 1911, and adopted by the Constituent Assembly as the national anthem on January 24, 1950. A full rendition of the anthem takes approximately 52 seconds, while a shortened version takes about 20 seconds.Protocols and UsageThe Ministry of Home Affairs (MHA) strictly governs the execution of both the National Anthem (Jana Gana Mana) and the National Song (Vande Mataram). At official and state functions, the National Song must always be played or sung first, followed by the National Anthem. Correct pronunciation, script, and diction are mandated for both compositions.Lyrics and Meaning (First Stanza)The lyrics of the anthem are taken from Tagore's larger hymn Bharoto Bhagyo Bidhata.
